# 300words

Write 300 words a day. Every day.

A minimalist daily-writing habit app. 300 words is about ten minutes — small enough to never skip, big enough to matter. Build the habit, clear your head, keep the streak alive.

## Landing page

The marketing site is a single self-contained `index.html` (inline CSS, no build step). It deploys as a static site.

### Local preview

Open `index.html` directly in a browser, or serve it:

```bash
npx serve .
```

### Deploy

Static site — deploys to Vercel (or any static host) with zero config. The primary CTA links to `/app`; repoint it when the app is live.
